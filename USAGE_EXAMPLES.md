## 常用技巧

### 1. 精确搜索
使用更具体的关键词可以缩小搜索范围：
- ❌ 搜索 "群" - 可能返回太多结果
- ✅ 搜索 "项目A" - 返回精确的群组

### 2. 批量发送
- 使用逗号分隔选择多个群组：`1,2,3,5,7`
- 使用 `all` 选择所有群组

### 3. 调整发送延迟
编辑 `~/.weibo-sender/config.json`：
```json
{
  "send_delay": 3  // 改为 3 秒延迟
}
```

### 4. 重新登录
如果遇到认证错误，删除配置文件重新登录：
```bash
rm ~/.weibo-sender/config.json
./weibo-sender
```
